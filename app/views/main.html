Intro

ConsistencyPlease is a web application that generates CSS for you basic styles

While working in the agency world I quickly realized that most designers (even the best) are usually unorganized and inconsistent in their mockups. Sometimes it's plain laziness or because of a time constraint; it can also be a result of bad work practices (such as not using a grid system); sometimes however, it's just because they tend to get lost in their creative thoughts and forget. It can be frustrating for a pixel-perfect freak like me but I came to accept this. In the end it's a litte bit more job because everything has to be double-checked but it also forced me to start using a thorough modular CSS structure, which is really good.

However I feel like some time could be saved by starting every projects with a solid, consistent CSS core. This is where this tool comes handy.

ConsistencyPlease allows anyone to generate CSS code with no knowledge whatsoever needed. You can then send the code (or file) to a front-end developer or use it yourself. Whether you need to replicate a mockup, want to test basic styles or simply seek inspiration, this application will jumpstart your project.

How to use:

  • Using the dashboard at the bottom fo the page, edit the default styles for the sections you want;
  • You can lock styles, add comments and visualize the code generated for a specfic element;
  • Once an element was edited, the modifications can be viewed in the (live) Preview section;
  • When everything looks good, Generate the base stylesheet template;
  • Copy/paste the generated CSS code or import the file in your project;
